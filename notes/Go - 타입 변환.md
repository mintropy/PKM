# Go - 타입 변환
- `strconv` 를 사용하여 변환

## 문자열 > 숫자
- `Atoi` : 문자열을 10진수 int 형으로
- `ParseInt` : 문자열을 특정 비트로 변환
	- `ParseInt("100" "10")`은 `Atoi("100")`과 같음

## 숫자 > 문자열
- `Itoa`
- `FormatInt`
