## Python
- os, pathlib를 활용한 파일 목록, 디렉터리 생성, 조회 등
	- 파일 목록 조회
		- `os.listdir("./data")`
		- 여기서 `/data` 로 접속하면 안되고, 시작 위치, .이나 C:// 등에서 해야함
	- 디렉터리 생성
		- `os.mkdir()`, `os.mkdirs()`
		- 단일 디렉터리면 mkdir, 아니라면 mkdirs로 생성가능
	- 파일 생성
		- `pathlib.Path().touch()`
		- Path의 파라미터로 생성할 파일위치 사용
		- 비어있는 json파일 생성시 파일을 읽어올 때 에러 발생
		- json 파일 생성은, `json.dump()`로 하는것이 좋음
