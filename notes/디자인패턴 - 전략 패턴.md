# 디자인패턴 - 전략 패턴
- 전략 패턴(strategy pattern) 또는 정책 패턴(policy pattern)
	- 실행 중에 알고리즘을 선택할 수 있게 하는 디자인 패턴
- 전략 패턴은
	- 특정한 계열의 알고리즘을 정의
	- 각 알고리즘을 캡슐화
	- 해당 계열 안에서 상호 교체가 가능하도록
