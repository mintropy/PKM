# JS - BigInt
- 다른 자료형들과 유사하게 사용할 수 있어 편리한 부분도 많은 것 같음
- 생성은 `BigInt()`를 통하여 생성하거나 정수 리터럴 뒤에 n을 붙여 생성할 수 있음

```javascript
const bigInt = bigInt(123)
const bigInt = 123n
```

- BigInt사이 연산은 일반적인 Number에서 연산처럼 진행할 수 있음
